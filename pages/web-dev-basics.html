Web Dev Basics

My journey into web development started with HTML and CSS, the fundamental tools for building and styling web pages. I learned how to structure content using HTML elements and bring designs to life with CSS, experimenting with layouts, colours, typography, and spacing. Understanding these basics gave me the ability to create visually appealing and functional web pages while setting a strong foundation for more advanced concepts like responsive design and interactivity. This stage of learning was all about getting comfortable with the core principles, and now, with these essential skills, I feel ready to take on more complex challenges in front-end development.

Understanding HTML: The Structure of the Web

HTML is the backbone of any webpage. It provides the basic structure using elements such as:

  • Headings (<h1> to <h6>) - Define section titles and page hierarchy.
  • Paragraphs (<p>) - Organise text content.
  • Links (<a>) - Connect pages and external resources.
  • Images (<img>) - Add visuals to enhance content.
  • Lists (<ul> and <ol>) - Structure items in bullet points or numbered sequences.
  • Divs and Spans (<div>, <span>) - Group and style content.

CSS: Bringing Web Pages to Life

Once the structure is in place, CSS is used to control the appearance of a webpage. Some of the key CSS concepts I learned include:

  • Selectors - Targeting elements (class, id, element selectors).
  • Colours & Typography - Customising text, fonts, and background colours.
  • Box Model - Understanding how margins, borders, padding, and content interact.
  • Positioning & Layouts - Using flexbox and grid to arrange elements.
  • Responsive Design - Making layouts adapt to different screen sizes using media queries.

Building My First Layouts

After learning the basics, I put my knowledge into practice by creating simple webpage layouts. This involved:

  • Structuring the page with semantic HTML elements (<header>, <nav>, <main>, <footer>).
  • Using CSS flexbox and grid to create responsive layouts.
  • Experimenting with colours, spacing, and typography to style content.
  • Ensuring the design looked good on different screen sizes with media queries.

Next Steps

Learning the basics of HTML and CSS was an important first step, but there's still so much more to explore. Next, I'll be diving deeper into:

  • JavaScript to add interactivity.
  • Advanced CSS techniques like animations and transitions.
  • Responsive and accessible design principles.
